Interesting Property

Let's let our encryption function be called f

\[\begin{aligned} f(x) = x^{3} \end{aligned} \]

Interesting Property

All we have done here is multiply, so we can do this

\[\begin{aligned} f(a * b) = (a * b)^{3} \end{aligned} \]

Interesting Property

All we have done here is multiply, so we can do this

\[\begin{aligned} f(a * b) = a^{3} * b^{3} \end{aligned} \]

Interesting Property

All we have done here is multiply, so we can do this

\[\begin{aligned} f(a * b) = f(a) * f(b) \end{aligned} \]

Interesting Property

We call functions with this property homomorphic over multiplication

\[\begin{aligned} f(a * b) = f(a) * f(b) \end{aligned} \]
